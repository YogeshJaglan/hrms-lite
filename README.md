## Overview
HRMS Lite is a lightweight Human Resource Management System built as a full-stack web application.  
It allows an admin to manage employee records and track daily attendance.

The project is designed to demonstrate end-to-end full-stack development including frontend, backend, database, and deployment.

## Live Demo
- Frontend: https://hrms-lite-five-sooty.vercel.app/
- Backend API: https://hrms-lite-mwb5.onrender.com
- API Test: https://hrms-lite-mwb5.onrender.com/api/employees

## Tech Stack
### Frontend
- React (Vite)
- JavaScript
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas (Cloud)

### Deployment
- Frontend: Vercel
- Backend: Render

## Features
- Add, view, and delete employees
- Mark daily attendance (Present / Absent)
- View attendance records per employee
- Data persistence using MongoDB
- Basic validations and error handling
- Clean and professional UI
